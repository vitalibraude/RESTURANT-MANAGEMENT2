## 🚀 הרצה מקומית

**דרישות מקדימות:** Node.js

1. התקן תלויות:
   ```bash
   npm install
   ```

2. הגדר את ה-`GEMINI_API_KEY` בקובץ `.env.local`:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

3. הרץ את האפליקציה:
   ```bash
   npm run dev
   ```

4. פתח בדפדפן: `http://localhost:3001`
